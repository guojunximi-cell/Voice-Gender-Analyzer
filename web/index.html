🦆 声音分析鸭 · 使用说明

这是什么？

上传或录制一段普通话或英文音频，分析声音的性别声学特征。输出整段均值 + 逐字音高/共鸣色彩，作声音训练的参考工具——并非判定。

分析流程

上传 / 录音
拖拽音频、选择文件，或调用麦克风（≤ 5 MB，< 3 分钟，中文 zh-CN 或英文 en-US）
▼
VAD 分段
Engine A · inaSpeechSegmenter K-3 神经网络分出语音 / 音乐 / 静音
▼
文本对齐
Engine C · 自由模式跑 ASR（FunASR / faster-whisper），跟稿模式直接用您的稿子；Montreal Forced Aligner 对齐到音素
▼
共振峰 + z-score
Praat 提 F1 / F2 / F3 → z-score 合成共鸣值
▼
三面板渲染
波形 · 中央三明治时间轴 · 右侧整段均值

如何使用？

  1. 拖拽音频 / 点击上传 / 录音（≤ 5 MB，< 3 分钟）
  2. 点击「开始分析」，等待鸭子跑完进度条
  3. 三块面板自动填充，无需额外点击

界面导览

上方波形
时间轴；拖动跳转播放，分段着色代表 AI 识别的语音 / 音乐 / 静音。
中央时间轴
音高色带 → 汉字 → 共鸣色带的三明治：三行按字严格同轴，每字一格槽位；字内多个音素按时长细分。点击色块或汉字跳转播放；← N/M → 翻页浏览全文。
右侧面板
整段音频的均值：基频、共鸣、F1/F2/F3 与音高范围参考条，底部为 AI 分类器的加权置信度。

色彩与区间

色彩方向
蓝色 = 男声方向（低音高 / 宽声道共鸣）；粉色 = 女声方向（高音高 / 窄声道共鸣）。
基频 F0
< 155 Hz 典型男声，155~185 Hz 中性区，> 185 Hz 偏女声。
共振峰 F2
< 1400 Hz 偏男，1600~1900 Hz 中性，> 2200 Hz 偏女。
共鸣（0~100%）
基于 F1/F2/F3 z-score 的合成分，越高越偏女声方向。

常见问题

音频有什么限制？

≤ 5 MB、< 3 分钟，常见格式（mp3 / wav / m4a / webm 等）。建议 ≥ 30 秒、安静环境单人朗读。

中英怎么切换？

上传前在右上角语言切换里选 zh-CN 或 en-US。语言决定示例稿件、ASR 模型、MFA 资源路径。

「自由模式」和「跟稿模式」有什么区别？

自由模式跑 ASR 自动出文本；跟稿模式直接用您粘贴的稿子，更快、对齐更稳定。短音频或方言/嘈杂环境推荐跟稿。

为什么有些字没显示？

MFA 对齐到那个字时置信度过低就会跳过；可以改用跟稿模式贴入完整稿子改善。

进度卡住或断开怎么办？

SSE 走 Redis Stream 流式回放，刷新页面可以重新接收，结果不会丢失。

数据会保留吗？

服务端不持久化音频与转写结果；浏览器会话结束即清空。

结果靠谱吗？

参考工具，非医学/法律/身份判定。模型有偏差，请结合主观感受与训练目标使用。

移动端能用吗？

支持触屏与录音；建议横屏查看时间轴；viewport 已锁，不会被双指缩放。
